## 手動テスト手順

1. **アプリケーションを起動**
   ```bash
   cd youtube-ai-thread
   npm run dev
   ```

2. **ブラウザでアクセス**
   - http://localhost:3000

3. **テストケース**

   **韓国語ビデオのテスト:**
   - 検索: "바이브 코딩"
   - ビデオ選択: "바이브 코딩 딱 5분만에 마스터 시켜드립니다"
   - 期待結果: 字幕抽出成功 → スレッド生成成功

   **英語ビデオのテスト:**
   - 検索: "javascript tutorial"
   - CC アイコン付きのビデオを選択
   - 期待結果: 字幕抽出成功 → スレッド生成成功

   **字幕なしビデオのテスト:**
   - 検索: "no captions video test"
   - 字幕のないビデオを選択
   - 期待結果: エラーメッセージ表示 → 手動入力オプション表示
